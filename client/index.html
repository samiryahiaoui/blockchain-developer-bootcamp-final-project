Metamask Detection

Borrower Input

  • First, Borrower must register by entereing their name (string) and click the rgister Borrower button.
  • Then the Borrower enters a loan amount (uint) and click the loan request button.
  • Borrowers must have distinct addresses.

Lender Input

  • First, Lenders must register by entereing their name (string) and click the rgister Lender button.
  • Lender can make bid on the available loans only.
  • Lender can find available loans by checking the button "Check available loans".
  • Then the Lender can make lending bids to registered loans.
  • In order to bid on a loan, Lenders enters loan_id (such as 0, 1, ..), an amount they are willing to commit (uint) and an interest rate (uint)

Anyone can Check Loan Status

  • Enter loan ID (0 for the first loan, ...) to get information on the loan.
